Advanced Loading Techniques
===========================

Whereas the basics of loading models has already been touched upon in the
:ref:`models-and-actors` section, Panda's asset loading system is very advanced,
supporting loads from arbitrary sources, a virtual file system, and asynchronous
(threaded) loading.  This section describes some of the possibilities in more
detail.
